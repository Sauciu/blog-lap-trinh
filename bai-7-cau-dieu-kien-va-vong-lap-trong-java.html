Bài 7: Câu điều kiện và vòng lặp trong Java

Hoàng Anh Thư tháng 12 23, 2025

Giới thiệu

Trong Java, câu điều kiện và vòng lặp giúp chương trình đưa ra quyết định và lặp lại các thao tác cần thiết. Dưới đây là những cấu trúc cơ bản thường dùng nhất.

Câu điều kiện if – else

if – else dùng để kiểm tra điều kiện đúng hoặc sai.

int number = 10;

if (number % 2 == 0) {
    System.out.println("So chan");
} else {
    System.out.println("So le");
}

Kết quả:

So chan

Câu lệnh switch – case

switch – case dùng khi cần so sánh một giá trị với nhiều trường hợp.

int day = 2;

switch (day) {
    case 1:
        System.out.println("Thu hai");
        break;
    case 2:
        System.out.println("Thu ba");
        break;
    default:
        System.out.println("Ngay khac");
}


Kết quả:

Thu ba

Vòng lặp for

for thường dùng khi biết trước số lần lặp.

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

Kết quả:

1
2
3
4
5

Vòng lặp while

while dùng khi chưa biết trước số lần lặp.

int i = 1;

while (i <= 3) {
    System.out.println(i);
    i++;
}


Kết quả:

1
2
3

Vòng lặp do – while

do – while đảm bảo khối lệnh chạy ít nhất một lần.

int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 3);

Kết quả:

1
2
3
