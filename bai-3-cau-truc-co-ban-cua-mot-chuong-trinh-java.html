Bài 3: Cấu trúc cơ bản của một chương trình Java

Hoàng Anh Thư tháng 12 13, 2025

Giới thiệu

Sau khi đã tìm hiểu tổng quan về Java, trong bài viết này mình sẽ giới thiệu cấu trúc cơ bản của một chương trình Java. Đây là kiến thức nền tảng mà người mới học Java cần nắm để hiểu và viết được những chương trình đầu tiên.

Một chương trình Java cơ bản gồm những gì?

Một chương trình Java đơn giản thường có các thành phần chính:
  • Class
  • Phương thức main
  • Các câu lệnh thực thi

Mỗi chương trình Java được lưu trong một file có phần mở rộng là .java.

Class trong Java

Trong Java, class là đơn vị cơ bản để xây dựng chương trình. Mọi đoạn mã đều phải nằm bên trong một class.

Ví dụ:

public class HelloJava {
}

Trong đó:

public: phạm vi truy cập

class: từ khóa khai báo lớp

HelloJava: tên class, phải trùng với tên file

Phương thức main

Phương thức main là điểm bắt đầu của chương trình Java. Khi chạy chương trình, Java sẽ tìm và thực thi phương thức này đầu tiên.

Cú pháp chuẩn của phương thức main:

public static void main(String[] args) {
}

Ý nghĩa:

public: cho phép chương trình được thực thi

static: chạy mà không cần tạo đối tượng

void: không trả về giá trị

String[] args: tham số dòng lệnh

Ví dụ chương trình Java hoàn chỉnh

Dưới đây là một chương trình Java đơn giản:

public class HelloJava {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

Khi chạy chương trình, màn hình sẽ hiển thị dòng chữ:

Hello Java
Giải thích nhanh chương trình

public class HelloJava: khai báo class

main: phương thức chính của chương trình

System.out.println(): dùng để in dữ liệu ra màn hình

{ }: xác định phạm vi của class và phương thức

;: kết thúc một câu lệnh

Việc hiểu rõ từng thành phần giúp mình tránh được các lỗi cú pháp khi mới học Java.

Quy tắc đặt tên cơ bản trong Java

Khi viết chương trình Java, mình cần lưu ý một số quy tắc sau:
  • Tên class viết hoa chữ cái đầu
  • Tên file phải trùng với tên class
  • Tên biến và phương thức viết thường chữ cái đầu
  • Không dùng dấu cách trong tên

Một số lỗi thường gặp

Khi mới học Java, mình và nhiều bạn thường gặp các lỗi như:

Quên dấu ;
Tên file không trùng với tên class
Viết sai cú pháp phương thức main
Thiếu hoặc dư dấu { }

Đây là những lỗi cơ bản nhưng có thể khiến chương trình không chạy được.
