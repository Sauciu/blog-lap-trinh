Câu lệnh điều kiện if – else trong JavaScript

Câu lệnh điều kiện cho phép chương trình đưa ra quyết định dựa trên điều kiện đúng hoặc sai. Đây là một trong những khái niệm cơ bản nhưng rất quan trọng trong lập trình, giúp chương trình “tư duy logic” và xử lý tình huống khác nhau.

1. Câu lệnh if

Câu lệnh if được sử dụng khi bạn chỉ cần kiểm tra một điều kiện. Nếu điều kiện trả về true, đoạn code trong khối sẽ được thực thi. 


let age = 20;

if (age >= 18) {
  console.log("Bạn đã đủ 18 tuổi");
}
// Kết quả dự kiến: Bạn đã đủ 18 tuổi

Ví dụ thực tế: kiểm tra xem người dùng có đủ tuổi truy cập một trang web hay không.

2. Câu lệnh if – else

Dùng else để xử lý khi điều kiện trong if là false. 


let score = 4;

if (score >= 5) {
  console.log("Bạn đã đậu");
} else {
  console.log("Bạn đã rớt");
}
// Kết quả dự kiến: Bạn đã rớt

3. Điều kiện so sánh

Các toán tử so sánh: >, <, >=, <=, ==, ===. Nên ưu tiên === để so sánh cả giá trị lẫn kiểu dữ liệu. 


let number = 10;

if (number === 10) {
  console.log("Giá trị đúng");
}
// Kết quả dự kiến: Giá trị đúng

Tip: "10" == 10 → true nhưng "10" === 10 → false. Dùng === để an toàn.

4. Câu lệnh else if

Khi muốn kiểm tra nhiều điều kiện theo thứ tự, dùng else if. 


let point = 8;

if (point >= 9) {
  console.log("Giỏi");
} else if (point >= 7) {
  console.log("Khá");
} else {
  console.log("Trung bình");
}
// Kết quả dự kiến: Khá

Ví dụ thực tế: xếp loại học sinh hoặc đánh giá mức độ thành công của một nhiệm vụ.
