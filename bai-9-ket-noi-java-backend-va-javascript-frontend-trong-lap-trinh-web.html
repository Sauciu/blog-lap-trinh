Bài 9: Kết nối Java Backend và JavaScript Frontend trong lập trình Web

Hoàng Anh Thư tháng 12 27, 2025

Đoạn code JavaScript (Frontend)

<button onclick="sendData()">Gửi yêu cầu</button>

<script>
function sendData() {
    fetch("http://localhost:8080/hello")
        .then(response => response.text())
        .then(result => {
            console.log(result);
        });
}
</script>


Giải thích ngắn:

fetch() dùng để gửi yêu cầu lên Server

Server trả dữ liệu về

console.log(result) dùng để xem kết quả nhận được

Kết quả hiển thị trên console:

Hello from Java Backend

Đoạn code Java (Backend)

@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response)
            throws IOException {
        response.getWriter().write("Hello from Java Backend");
    }
}


Giải thích ngắn:

  • Servlet nhận yêu cầu từ JavaScript
  • Xử lý và trả dữ liệu về cho Client
  • Dữ liệu trả về là chuỗi văn bản
