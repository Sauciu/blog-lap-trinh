Bài 5: JavaScript là gì? Vai trò của JavaScript trong lập trình Web

Hoàng Anh Thư tháng 12 17, 2025

Giới thiệu

Sau khi đã học các kiến thức cơ bản về Java, trong bài viết này mình bắt đầu làm quen với JavaScript. Đây là ngôn ngữ giúp website có thể tương tác với người dùng và trở nên sinh động hơn. JavaScript thường được sử dụng ở Frontend và chạy trực tiếp trong trình duyệt.

JavaScript là gì?

JavaScript là ngôn ngữ lập trình dùng để xử lý các hành động trên trang web như click chuột, nhập dữ liệu và thay đổi nội dung hiển thị.

Ví dụ hiển thị thông báo:

alert("Hello JavaScript");

Kết quả:

Trình duyệt hiển thị hộp thoại:

Hello JavaScript

Khai báo biến trong JavaScript

JavaScript dùng let để khai báo biến.

let name = "JavaScript";
let age = 20;

console.log(name);
console.log(age);


Kết quả (Console):

JavaScript
20

Xử lý sự kiện người dùng

JavaScript thường được dùng để xử lý các sự kiện như click chuột.

<button onclick="sayHello()">Click me</button>

<script>
function sayHello() {
    alert("Xin chào!");
}
</script>

Kết quả:
Khi bấm nút Click me, trình duyệt hiển thị:

Xin chào!

Thay đổi nội dung trang web

JavaScript có thể thay đổi nội dung HTML mà không cần tải lại trang.

Nội dung ban đầu

<p id="text">Nội dung ban đầu</p>

<script>
document.getElementById("text").innerHTML = "Nội dung đã thay đổi";
</script>


Kết quả:
Dòng chữ hiển thị trên trang đổi thành:

Nội dung đã thay đổi

So sánh đơn giản

let a = 5;
let b = 10;

console.log(a < b);

Kết quả (Console):

true
