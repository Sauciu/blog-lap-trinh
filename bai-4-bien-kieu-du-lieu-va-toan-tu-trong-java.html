Bài 4: Biến, kiểu dữ liệu và toán tử trong Java

Hoàng Anh Thư tháng 12 13, 2025

Giới thiệu

Sau khi đã làm quen với cấu trúc cơ bản của một chương trình Java, trong bài viết này mình sẽ tiếp tục tìm hiểu về biến, kiểu dữ liệu và toán tử. Đây là những kiến thức nền tảng giúp chương trình lưu trữ và xử lý dữ liệu.

Biến trong Java

Biến dùng để lưu trữ dữ liệu trong chương trình. Khi khai báo biến, mình cần xác định kiểu dữ liệu, tên biến và giá trị.

Cú pháp khai báo biến:

kieudulieu tenBien = giaTri;


Ví dụ:

int age = 20;
double score = 8.5;

Kiểu dữ liệu trong Java

Một số kiểu dữ liệu cơ bản thường dùng

int number = 10;
double point = 9.0;
boolean isPassed = true;
char grade = 'A';


Ngoài ra, Java còn có kiểu String dùng để lưu chuỗi ký tự:

String name = "Java";

Toán tử trong Java

Toán tử số học
int a = 10;
int b = 3;

System.out.println(a + b); // cộng
System.out.println(a - b); // trừ
System.out.println(a * b); // nhân
System.out.println(a / b); // chia
System.out.println(a % b); // chia lấy dư
Toán tử so sánh
int x = 5;
int y = 10;

System.out.println(x == y);
System.out.println(x < y);
System.out.println(x > y);
Toán tử logic
boolean p = true;
boolean q = false;

System.out.println(p && q);
System.out.println(p || q);
System.out.println(!p);
Ví dụ chương trình minh họa
public class Example {
    public static void main(String[] args) {
        int a = 5;
        int b = 10;
        int sum = a + b;

        System.out.println("Tong = " + sum);
    }
}

Chương trình trên thực hiện phép cộng hai số và in ra kết quả.

Phải khai báo biến trước khi sử dụng

Không thể gán sai kiểu dữ liệu cho biến

Phân biệt = (gán) và ==(so sánh)

Đặt tên biến rõ ràng, dễ hiểu
