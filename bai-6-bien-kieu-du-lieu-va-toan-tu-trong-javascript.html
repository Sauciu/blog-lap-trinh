Bài 6: Biến, kiểu dữ liệu và toán tử trong JavaScript

Hoàng Anh Thư tháng 12 18, 2025

Giới thiệu

Trong bài viết này, mình tiếp tục làm quen với biến, kiểu dữ liệu và toán tử trong JavaScript. Đây là những kiến thức cơ bản giúp JavaScript lưu trữ và xử lý dữ liệu.

Biến và kiểu dữ liệu trong JavaScript

JavaScript là ngôn ngữ kiểu động, nên không cần khai báo kiểu dữ liệu trước. Hiện nay, mình thường dùng let và const để khai báo biến.

let name = "JavaScript";
let age = 20;
const PI = 3.14;

console.log(name);
console.log(age);
console.log(PI);

Kết quả (Console):

JavaScript
20
3.14

Toán tử trong JavaScript

JavaScript hỗ trợ các toán tử số học, so sánh và logic.

let a = 5;
let b = 10;

console.log(a + b);      // cộng
console.log(a < b);      // so sánh
console.log(5 === "5");  // so sánh cả giá trị và kiểu

Kết quả (Console):

15
true
false
